# AI-Powered Indian Sign Language Translator

## Overview

The AI-Powered Indian Sign Language Translator is a real-time system that recognizes Indian Sign Language (ISL) hand gestures and converts them into text. The project aims to improve communication accessibility for hearing and speech-impaired individuals.

## Features

* Real-time hand gesture detection
* Sign language to text conversion
* User-friendly interface
* Responsive frontend interface
* Gesture recognition using computer vision

## Technologies Used

* React.js
* HTML5
* CSS3
* JavaScript
* Python
* OpenCV
* TensorFlow
* MediaPipe
* NumPy

## Project Workflow

1. Capture live video using a webcam.
2. Detect hand landmarks using MediaPipe.
3. Process gesture data using machine learning models.
4. Convert recognized gestures into text output.
5. Display results through a React.js-based user interface.

## Applications

* Communication assistance for hearing and speech-impaired individuals
* Educational institutions
* Accessibility solutions
* Human-computer interaction systems
